# Browsing Apps

Ciyex Hub makes it easy to discover apps that fit your practice needs.

## Accessing Ciyex Hub

1. Log in to your Ciyex EHR
2. Click the **Ciyex Hub** icon (Store icon) in the left sidebar
3. You'll land on the Hub browse page

## Searching for Apps

Use the **search bar** at the top of the Hub page to find apps by:

- App name (e.g., "telehealth", "billing")
- Category (e.g., "clinical", "operations")
- Feature keywords (e.g., "SMS", "claims", "AI")

## Filtering by Category

Click on category pills to filter the app list:

- **All** — Show all apps
- **AI** — Artificial intelligence tools
- **Billing** — Payment and billing apps
- **Clinical** — Clinical decision support
- **Clinical Reference** — Medical code libraries
- **Communication** — Messaging platforms
- **Infrastructure** — Storage, compute, networking
- **Operations** — Credentialing, HR, admin
- **Platform** — Low-code tools, customization
- **Revenue Cycle** — Claims, denials, payments
- **Telehealth** — Video consultations

## Featured Apps

Apps marked as **Featured** appear at the top of the browse page. These are recommended apps that are widely used and highly rated.

## App Cards

Each app card shows:

- **Icon** — Visual identifier
- **Name** — App name
- **Category** — What type of app it is
- **Description** — What the app does
- **Key Features** — Top capabilities
- **Pricing** — Free, per-use, or subscription
- **Installed Badge** — Shows if you've already installed it

## App Detail Page

Click on any app card to see the full detail page with:

- Complete description and feature list
- Pricing plans and trial information
- FHIR resources the app uses
- Configuration options
- Install/Uninstall button
- Reviews and ratings
